# Comet Runner ☄️

A fast-paced space arcade game built using pure HTML, CSS, and JavaScript.

## 🎮 Gameplay
Dodge incoming obstacles, collect glowing energy orbs, and survive as long as possible while the speed keeps increasing.

## 🕹️ Controls
- A / D or Arrow Keys → Move left/right
- Space → Boost speed
- Mobile → Touch controls available

## ⚡ Features
- Smooth canvas-based gameplay
- Dynamic difficulty scaling
- Particle effects & screen shake
- High score saving
- Mobile-friendly controls

## 📁 Project Structure
index.html → Game UI  
style.css → Styling  
script.js → Game logic
